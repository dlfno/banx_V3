# Simulador de Debate Banxico

Simulador multi-agente de la Junta de Gobierno del Banco de México.

## Objetivos

- Agentes con distintas posiciones (hawkish/dovish/centrista, etc.) que debaten entre sí.
- Sistema de votación y generación automática de minutas.
- Agentes con acceso a herramientas y búsqueda web.
- Backend conectado a OpenRouter o Anthropic (LangChain opcional).
- Interfaz gráfica en React.
- Dos modos de operación:
  - **Chat 1-a-1** con un agente individual.
  - **Simulación de junta** completa con debate y votación.
- Memoria persistente compartida entre ambos modos.
